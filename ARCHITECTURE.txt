═══════════════════════════════════════════════════════════════════════════
                            DATA FLOW SEQUENCE
═══════════════════════════════════════════════════════════════════════════

Step 1: User Triggers
   └─> Web Dashboard → Click "Generate Labels" button

Step 2: API Request
   └─> GET /api/generate-labels → Server receives request

Step 3: Fetch from Zoho
   └─> ZohoBooksAPI.getTomorrowDeliveries()
       ├─> Query Zoho Books API
       ├─> Filter: cf_delivery_date = tomorrow & status = sent
       └─> Returns: Array of invoice objects

Step 4: Extract Data
   └─> For each invoice:
       ├─> customer_name
       ├─> shipping_address (street, city, state, zip)
       ├─> phone
       └─> invoice_number

Step 5: Generate Excel
   └─> DeliveryLabelGenerator.generateLabels(invoices)
       ├─> Create workbook with ExcelJS
       ├─> Set A4 page dimensions
       ├─> Set column widths (99.1mm equiv)
       ├─> Set row heights (38.1mm equiv)
       ├─> For each delivery:
       │   ├─> Merge cells (6 rows × 1 col)
       │   ├─> Format text (bold name, normal address)
       │   ├─> Add border (optional)
       │   └─> Position in 2×7 grid
       └─> Save to: output/Delivery_Labels_TIMESTAMP.xlsx

Step 6: Download
   └─> Server responds with:
       ├─> Success message
       ├─> File download URL
       └─> Auto-downloads to user's device

Step 7: Print
   └─> User opens Excel file
       ├─> Print settings: A4, 100% scale
       └─> Prints on Avery 5162 label sheets


═══════════════════════════════════════════════════════════════════════════
                          TECHNOLOGY STACK
═══════════════════════════════════════════════════════════════════════════

Backend:
   ├─> Runtime:      Node.js 18+
   ├─> Framework:    Express.js 4.x
   ├─> Excel Gen:    ExcelJS 4.x
   ├─> HTTP Client:  Axios (Zoho API)
   ├─> Date Utils:   date-fns
   └─> Config:       dotenv

Frontend:
   ├─> HTML5 + Vanilla JavaScript
   ├─> CSS3 (gradients, flexbox)
   └─> No framework (keep it simple)

External APIs:
   ├─> Zoho Books API v3
   │   └─> Endpoints: /invoices, /invoices/{id}
   └─> OAuth 2.0 authentication

Hosting Options:
   ├─> Local:  Windows/Mac/Linux + PM2
   ├─> Cloud:  DigitalOcean, Render, Vercel
   └─> Port:   3000 (configurable)
